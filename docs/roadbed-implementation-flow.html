F1 2026 Track System

Roadbed First Implementation Flow

目標不是讓柏油材質更像道路，而是先讓道路地基嵌入並改寫地形。terrain 必須尊重 roadbed；地基不能像貼片一樣浮在山體上。地基、視覺地形、Rapier heightfield、懸吊 sample、surface 判定必須看見同一份 stamped terrain。

現在的問題

道路目前像一片貼在地形上的 mesh。原始山體仍留在道路下面，導致山體可能穿過路中，或讓懸吊與車身防穿透被錯誤地形高度干擾。這不是柏油不夠厚，而是地基沒有先切入 terrain。

新的原則

先從 SVG/editor path 建立 road corridor，再把 roadbed stamp 進 terrain heightmap。terrain 是被改造後的結果，柏油 mesh 只是鋪在已嵌入的地基上。

  1. Terrain 先被 roadbed cut/fill，視覺與物理都讀同一份。
  2. 柏油僅薄鋪面，跟著已嵌入的 roadbed 走，不再蓋住山體。
  3. Track Limit、curb 用同一份 corridor boundary。
  4. 懸吊先讀 surface query，再才 Rapier raycast / terrain fallback。
  5. 使用者看到的是 Track Limit；儲存格式仍為 edge_line。

實作階段

1

建立 Road Corridor

從現有 track_ribbon 或 SVG/editor path 產生道路佔用區，作為所有後續系統的共同輸入。

centerline samples
road width
shoulder width
falloff width
2

Roadbed Stamping

對 heightmap grid cell 計算到道路中心線的距離，把道路核心切入或填起成地基高度，路肩和 falloff 平滑接回山體。

core: road height
shoulder: softened edge
falloff: smoothstep blend
outside: original terrain
3

統一 Terrain / Physics

getHeightAt()、視覺地形、Rapier HeightfieldCollider、懸吊 body penetration sample 都改讀 stamped terrain。

TerrainGround visual
HeightfieldCollider
Suspension fallback
Debug height probes
4

鋪 Asphalt Mesh

柏油 mesh 放在已處理好的 roadbed 上，只負責薄鋪面、厚度側邊、UV 與材質，不再負責蓋住山體。

top surface +0.035m
thin side wall
arc-length UV
pit road material zone
5

Surface Query

新增座標查詢，回傳車輪所在位置的高度、法線、材質與距離中心線資訊。React 先用它同步 Rust surface enum。

queryTrackSurface(x, z) => {
  height,
  normal,
  material,
  distanceFromCenter,
  roadT
}
6

Suspension Raycast 修正

懸吊支撐高度順序改為 road/surface query、Rapier raycast、stamped terrain fallback，避免原始山體重新把車頂起來。

road surface first
Rapier hit second
terrain fallback
normal sync to WASM
7

Curbs / Painted Areas / Road Edge

curb、track limit、painted area 全部依附 road corridor boundary，不再各自重新猜道路邊界。

curb follows edge
painted area on asphalt
track limit shared frames
patch affects material only
8

驗證與品質門檻

測試重點是高度一致、沒有尖峰、山坡穿過道路時 core 仍平順、falloff 自然接回原地形。

roadbed unit tests
height consistency tests
suspension tolerance tests
visual browser check

第一個 Commit 範圍

新增 roadbed derived terrain layer，讓 track_ribbon 可以把道路 footprint stamp 進 heightmap。 先不改 shader、不改 Rust、不做厚柏油，只先保證路中間不會有山體突出。

暫時不做

不先重寫車輛物理，不先做高級柏油材質，不先處理所有 curb 細節。這些都排在地基穩定之後。

完成判定

視覺

道路核心沒有山體穿出，路肩自然接回地形。

物理

Rapier heightfield 與 getHeightAt() 對同一座標回到一致高度。

車感

車輪從山坡進入道路時不跳車，懸吊沒有被原始山體頂起。
